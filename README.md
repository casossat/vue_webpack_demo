# vue_webpack_demo
Simple app that implement webpack + Vuejs

```
//generate package.json
npm init 

//generate tsconfig.json
tsc --init

npm install
```

**Note** Use nvm > 8.0 

See webpack.config.js

## Include
- pug
- vutify
- ts
- sass

## deployment 

``
npm build
``

## build

``
npm start
``
